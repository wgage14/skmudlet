# Create new Mudlet profile
Profile name: whatever-you-want
Server addr: mud.shatteredkingdoms.org
Port 1996

# Scripts
Import skmudlet.mpackage using Toolbox --> Package Manager --> Install new Package.

# ID table
Import pre-populated lua tables (in tables folder) to %userprofile%\.config\mudlet\profiles\(whatever-you-named-your-sk-profile). You can do this after loading the mpackage but it requires closing/reopening mudlet to use
